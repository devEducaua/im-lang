# IM language
Indu-Markup or Improved Markup language is a markup language similar to Markdown and Gemini.

```im
- Title Page -

- IM lang
normal text.
*italic text*.
**bold text**.
***italic and bold text***.

-- todo-lists
[ ] todo item
[x] todo item

-- tables
id | extension
1  | im
2  | md
3  | gmi
```

IM is still in development.
